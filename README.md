# Vision_Android_con_filtros
# 📱 Visión por Computador en Android con Filtros

Aplicación móvil desarrollada en Android que implementa técnicas de **visión por computador** utilizando la librería OpenCV. Permite capturar video en tiempo real y aplicar diferentes filtros de procesamiento de imágenes.

---

## 🚀 Características

* 📷 Captura de video en tiempo real desde la cámara
* 🎨 Aplicación de filtros de imagen
* ⚙️ Procesamiento de imágenes en vivo
* 🔍 Uso de algoritmos básicos de visión por computador

---

## 🛠️ Tecnologías utilizadas

* Java / Kotlin
* Android Studio
* OpenCV (Open Source Computer Vision Library)

---

## 📂 Estructura del proyecto

```
Vision_Android_con_filtros/
│── app/                    # Aplicación Android
│── openCV/                 # Módulo SDK de OpenCV
│── gradle/                 # Archivos de configuración de Gradle
│── build.gradle.kts        # Script de construcción raíz
│── settings.gradle.kts     # Configuración de subproyectos
│── opencv-4.12.0-android-sdk/ # SDK externo de OpenCV
```

---

## ⚙️ Requisitos

* Android Studio instalado
* SDK de Android configurado
* Dispositivo físico o emulador Android
* OpenCV integrado en el proyecto

---

## 📸 Funcionalidades implementadas

* Conversión a escala de grises
* Detección de bordes
* Aplicación de filtros personalizados
* Procesamiento en tiempo real

---

## ⚠️ Notas importantes

* El SDK de OpenCV puede no estar incluido en el repositorio debido a su tamaño.
* Asegúrese de configurarlo correctamente en Android Studio si es necesario.
